# Test Plan

## Objective
The objective of this test  plan is to ensure that the financing application web system functions correctly according to the defined requirements and is ready to realease.

## Scope
The following features will be tested:
- Login functionality for all actors
- Financing application submission by Users
- Financing application review and approval by Manager
- Data management by Admin
- Data validation
- Basic CRUD operations

## Test Type
- Manual Testing
- Functional Testing

## Actors Involved
- User
- Manager
- Admin

## Test Environment
- Application Type : Web Application
- Browser : Google Chrome
- Test Environment : Local / Staging

## Test Tools
- Google Sheets (test case documentation)
- GitHub (test documentation)

## Entry Criteria
- Application build is deployed
- Test Environment is accesible
- Test accounts for User, Manager, and Admin are available

## Exit Criteria
- All critical test cases are executed
- No high or critical secerity bugs remain open
- Application is ready for User acceptance testing
