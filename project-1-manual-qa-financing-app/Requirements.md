# Application Requirements

## Application Name
Financing Application Web System

## Application Type
Web Application

## Description
This web-based application allows users to submit financing applications online. Users can log in, fill out a financing application form, submit the application, and view the submission status. Admin users can manage and review submitted financing applications.

## Actors
- User
- Manager
- Admin

## Flow Utama
- User accesses the web application through a browser and logs in using valid credentials. After successful login, the user fills in the financing application form and submits the application. The system validates the input data and stores the submission.

- Manager logs into the system to review submitted financing applications and performs approval or rejection actions based on predefined criteria.

- Admin manages user accounts, application data, and system configuration through the admin dashboard.
 The system validates the input data and stores the submission. Admin users can view and manage submitted financing applications through the admin dashboard.

## Scope of Testing
- Login functionality
- Financing application submission (simple form)
- Data validation
