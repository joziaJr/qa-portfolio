# Test Scenarios

## User Scenarios
- Verify user can log in with valid credentials
- Verify user cannot log in with invalid credentials
- Verify user can access financing application form
- Verify user can submit financing application with valid data
- Verify system validates required fields on financing application form
- Verify submitted financing application is saved and displayed correctly

## Manager Scenarios
- Verify manager can log in with valid credentials
- Verify manager can view submitted financing applications
- Verify manager can approve a financing application
- Verify manager can reject a financing application
- Verify application status is updated after manager action

## Admin Scenarios
- Verify admin can log in with valid credentials
- Verify admin can view all financing application data
- Verify admin can create new data records
- Verify admin can update existing data records
- Verify admin can delete data records
